# Graphics Application
A graphics application framework...

## Install GLFW
```
brew install glfw
```

## Install CMAKE
```
brew install cmake
```

## Install Vulkan
go to website to get the SDK

## Install CGLM (for math in c)
```
brew install cglm
```

## Install OpenAL
```
brew install openal-soft
```

## Compiling Shaders
```
glslangvalidator -V shaders/basic.frag -o shaders/basic.frag.spv
```
